Changelog

Release notes for CellVision Desktop (Open Source).

Version 2.4

March 2026 Latest
  • Materials Li-PbO has been removed as an anode choice due to insufficient confidence in available literature.
  • Materials NMC811 half-cell capacity at 4.4V vs Li decreased from 214 to 210 mAh/g.
  • Materials Additional anode binders added: PAA, CMC, and SBR.
  • Materials Manganese-based Potassium Prussian White cathode added.
  • Materials Hard carbon capacity and voltage curves have been updated to reflect material improvements in the market.
  • Materials Sodium Prussian White cathode capacity and discharge curve updated.
  • Bug Fix Fixed the volume change calculation instability.
  • Bug Fix Fixed a bug that occasionally broke cell costs by removing a circular calculation.
  • Bug Fix Fixed a slight calculation error with anode porosity.
  • Bug Fix Blended anode porosity in the discharge state is now 20% instead of 10% to match the other anodes.
  • Bug Fix The anode current collector for potassium-ion batteries is now correctly set to aluminum instead of copper.
  • Bug Fix Volume change calculation for lithium-sulfur cells has been corrected.
  • Bug Fix Metal anodes no longer incorrectly include binder or carbon content.
  • Bug Fix Current collector foil mass was incorrectly halved and has been corrected. This increases the mass of all cells.
  • Prices Active and inactive material prices have been updated.
  • Feature Carbon anode porosity is now dependent on the "energy" or "power" cell selection.
  • Feature Anode binder density is now dependent on anode type.
  • Feature Anode first cycle efficiency (FCE) has been added, which reduces cell capacity and adjusts the lower voltage cutoff of the cell.
  • Feature Target cell loading is now based on both an upper manufacturing thickness limit and a target areal current density instead of the previous loading override option.
  • Feature You can now set a target Ah for the cell instead of a pre-defined cell dimension. This is more appropriate for comparing different chemistries.
  • Feature Anode-free cell modeling has been improved: cells are now built without lithium metal and volume change during cycling is accounted for separately.

Version 2.3

October 2025
  • UI Emphasized the need to enable iterative calculations for the spreadsheet to work properly.
  • Materials Added sulfur cathode, sodium metal anode, and lithium metal anode.
  • Materials Updated hard carbon to Kuranode Type 2, effectively increasing nominal voltage for all sodium hard carbon cells and improving SIB energy density.
  • Bug Fix Fixed a circular calculation bug that caused blended anodes to give unstable results.
  • Bug Fix Hard carbon volume change calculation was referencing graphite capacity instead of hard carbon. Minimal impact to results.
  • Bug Fix Incorporated percent of lithiation into alloy anode volume change calculation, effectively decreasing the volume change of all alloy anodes.
  • Bug Fix Fixed the silicon anode error by adding a Si anode charge curve.
  • Inputs Changed alloy anode porosity in the charged state to a minimum of 20% (previously 10%) to allow for electrolyte diffusion. Alloy anode volume increased accordingly.
  • Inputs Decreased SiO anode capacity from 3,000 to 1,650 mAh/g.
  • Prices Updated cell component costs based on Shanghai Metals Market rates.

Version 2.2

June 2025
  • Bug Fix Production scaling factors were incorrectly linked from BatPaC. The fix has slightly increased cell costs.
  • Bug Fix Electrode coating area parameter missed a layer count multiplier when calculating electrode manufacturing costs. The fix has increased cell manufacturing costs.
  • Bug Fix Anode foil cost did not automatically update to aluminum when a sodium anode was selected. The fix slightly decreased foil costs for sodium batteries.
  • Prices Copper foil costs were substantially underestimated and have been updated accordingly.
